Fragment A:
  Base URL: https://api.athenaeum.example/v2
  Auth: Bearer token required
  Token hint: base64 decode "Q29kZXgtVG9rZW4tNzM5"

Fragment B:
  Resource path pattern: /fragments/{fragment_id}
  fragment_id format: realm-code + "-" + sequence
  Example realm codes: fmt, api, pat, asm, log

Fragment C:
  Query parameters:
    - decode: boolean (default false)
    - verify: boolean (default false)
    - format: enum [raw, json, xml]

Fragment D:
  HTTP Methods:

    - POST /fragments/{id}/decode — submit decoding attempt
    - PUT /fragments/{id}/verify — verify fragment integrity

Fragment E:
  Headers required:
    - Authorization: Bearer <token>
    - X-Athenaeum-Realm: <realm_name>
    - Accept: application/json

Fragment F:
  Success response format:
    {
      "fragment_id": "...",
      "status": "verified|decoded|pending",
      "checksum": "sha256:...",
      "content": "..."
    }
