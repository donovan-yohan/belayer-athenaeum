# The Summons Pedestal

A marble pillar rises from the center of the Vestibule. Atop it rests a parchment, its text obscured by some encoding. A faint magical aura surrounds it — the message was clearly meant to survive the shattering, but only for those who know how to read it.

The parchment appears to contain a base64-encoded message, but something about the encoding feels... layered. Perhaps the heroes must peel back the obfuscation to reveal the true summons.

## What You See

A file `encoded_summons.txt` rests on the pedestal. It contains a long string of seemingly random characters.
